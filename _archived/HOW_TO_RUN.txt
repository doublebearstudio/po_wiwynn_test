Step 2: File > Open > po_wiwynn_test_v0002.usda

Step 4: Open file: script_editor_simple.py
Step 5: Copy ALL contents (Ctrl+A, Ctrl+C)
Step 6: Paste into Script Editor (Ctrl+V)
Step 7: Click "Run" button (or Ctrl+Enter)
Step 8: Watch the robots work!

================================================================================
                         TROUBLESHOOTING
================================================================================

Problem: Script doesn't run
Solution: Make sure USD file is loaded in Isaac Sim

Problem: Robots don't move
Solution: Check that physics is enabled and simulation is playing

Problem: Objects fall through
Solution: Verify collision detection is enabled in USD file

Problem: Need to adjust positions
Solution: Edit config.py (standalone) or script values (Script Editor)
